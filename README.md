# bitcoin-address
A small Rust CLI (and library) to generate a `p2pkh` bitcoin address from a given private key. It also contains a possibility to work with `p2sh` addresses (WIP).

## Usage

### Run CLI
```bash
cargo run -- <PRIVATE_KEY>
```
*Use `-v` for verbose output*

### Test
```bash
cargo test
```
